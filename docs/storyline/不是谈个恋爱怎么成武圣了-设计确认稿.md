## 一、创作核心

### 双线结构

- 第一主线：陆寻围绕九段武道体系持续晋级。
- 第二主线：陆寻与苏晚晴从相识、合作到恋爱的完整感情过程。
- 每一卷都必须同时有明确的段位进展和感情进展。
- 第一卷结尾，陆寻与苏晚晴正式确认恋爱关系。

### 主角反差

- 陆寻是纯粹的武道天才，不依赖系统、宝物或复制型能力。
- 他能迅速理解武道原理、修正动作，并在实战中高速适应。
- 他在武道上极度敏锐，在感情上不擅长判断和表达。
- 感情喜剧来自“看得懂拳，看不懂苏晚晴”，但不能长期依靠低级误会拖延剧情。

## 二、九段武道体系

官方采用一至九段。每段只有一个明确境界，不设初期、中期、巅峰等小境界。

九段分为下三段、中三段、上三段。跨入六段是全体系最重要的质变分水岭。

| 官方段位 | 境界俗称 | 核心能力 |
| --- | --- | --- |
| 一段 | 鸣骨境 | 筋骨第一次蜕变，力量、速度脱离普通人范围 |
| 二段 | 合劲境 | 全身力量贯通，形成完整发力 |
| 三段 | 炉息境 | 体内形成稳定循环，恢复力与持续作战能力质变 |
| 四段 | 武师境 | 真力离体，可以隔空伤人 |
| 五段 | 罡身境 | 真力贴体形成罡甲，防御能力质变 |
| 六段 | 宗师境 | 武道意志外放，在身后凝聚巨大的武道真身 |
| 七段 | 天行境 | 武道真身由虚转实，可以托举本体飞行并借用天地力量 |
| 八段 | 武王境 | 展开城市级武道领域，属于国家核心战略力量 |
| 九段 | 武圣境 | 国家级力量，武道真身化为圣相，足以改变国家战争结果 |

### 武道真身

- 每位宗师的武道真身都不同。
- 真身形态由武者自己的武道道路、性格和核心信念决定。
- 六段以后按以下视觉路线成长：真身显化、真身化实、真身成域、真身化圣。
- 六段宗师显化武道真身，应当成为全书第一次世界观级震撼。

### 高段武者数量

- 六段宗师境：普通城市约数百人，核心大城市可达数千人。
- 七段天行境：每座城市约数十人。
- 八段武王境：每个国家只有个位数。
- 九段武圣境：全球寥寥无几，当前时代也可能没有。

### 基础力量层次

- 气血：人体生命与体能基础，一至二段主要修炼气血。
- 真息：三段炉息境形成的体内稳定循环。
- 真力：真息经过身体与武道训练转化的战斗力量，四段开始可以离体。
- 罡力：五段将真力稳定覆盖身体，形成攻防一体的罡身。
- 以上属于同一套力量在不同段位的形态，不再设置灵力、斗气等平行能源。

### 五种真力劲性

- 刚劲：正面爆发与强破坏。
- 柔劲：卸力、牵引和改变力量方向。
- 透劲：穿过外层防御，攻击内部。
- 震劲：连续震荡，破坏身体或物体结构。
- 缠劲：黏住对方真力，限制动作与力量运转。
- 武者通常选择一种主劲性和一种辅劲性，也可以通过长期修炼形成组合。
- 真力劲性取代传统火、冰、雷等元素属性，保持高武战斗质感。

### 武器与战技

- 拳脚与兵器没有绝对高低，各自拥有距离、爆发、变化和持续作战优势。
- 常见路线包括拳、掌、擒拿、剑、刀、枪、棍和弓。
- 战技按功能分为基本式、身法、攻伐技、护体技、绝技、秘法和禁式。
- 招式可以存在熟练度，但招式熟练度不属于段位小境界。
- 拳意、剑意、刀意不是独立能源，而是武者个人道路和战斗意志的具体表现。
- 当前阶段不提前确定陆寻个人战技、绝招、秘法和禁式的名称。
- 个人招式名称必须等分卷大纲确定后，结合该卷对手、突破场景、人物经历和武道成长自然命名。
- 禁止为了填技能表而提前堆砌招式名称。

### 个人招式的正常成长层级

- 一段主要学习基本式。
- 二段学习成熟战技，并形成个人战斗习惯。
- 三段通常只能理解和修改现有战技；陆寻会提前创造第一门个人战技，以此体现天才程度。
- 四段武师开始普遍自创战技；陆寻在这一阶段进一步创造个人绝招或秘法。
- 五段围绕罡身建立攻防体系，陆寻会创造能衔接未来武道真身的个人秘法。
- 六段将此前的战技、绝招和秘法整合成完整个人道路，并凝聚对应的武道真身。
- 七段以上，个人武道继续发展为天行手段、王域能力和最终圣法。
- 以上只确认能力层级与成长逻辑，具体能力和名称随分卷大纲逐卷确定。

### 段位战斗压制

- 四段真力离体，三段武者很难安全接近。
- 五段罡身形成稳定防御，四段武者很难正面破防。
- 六段显化武道真身，五段正面对抗几乎必败。
- 七段可以飞行并借用天地力量。
- 八段展开王域，形成城市级范围压制。
- 九段圣相能够影响国家级战场。
- 越级胜利只能依靠多人合作、地形克制、提前布局或特殊代价，不能让陆寻长期无视段位差距。

### 舞台升级原则

- 大学阶段主要接触一至四段。
- 五段是全国大学生中的顶点。
- 六段之前，故事舞台以校园、城市赛事和大学联赛为主。
- 主角进入六段后，故事舞台扩大到国家级事件。
- 七段开始涉及城市级危机。
- 八段进入国家之间的战略博弈。
- 九段影响全球格局，并终结可能存在的“无圣时代”。

### 大学阶段与职业转折

- 第1-70章（暂定）：大一，陆寻从未入段进入二段，完成天赋曝光、跨院培养和校队立足；第一卷结尾与苏晚晴确认恋爱关系。
- 第71-150章（暂定）：大二，陆寻从二段进入三段，参加城市及区域大学联赛。
- 第151-230章（暂定）：大三，陆寻从三段进入四段，进入全国大学联赛核心赛场。
- 第231-310章：大四，陆寻从四段进入五段，成为全国大学生武道顶点，并完成大学篇的最终赛事。
- 第310章前后：陆寻毕业，正式签约职业队，大学篇结束，职业篇开始。
- 第311章以后：陆寻以五段新人身份进入职业赛场，五段到六段的宗师突破必须在职业篇完成。
- 六段宗师是大学赛场与国家级舞台之间的分界，不能在大学阶段提前完成。

以下章节范围已经确认，后续分卷大纲和细纲必须严格遵守：

| 章节范围 | 段位进程 | 阶段结果 |

| 第71-150章 | 二段到三段炉息境 | 完成下三段成长 |
| 第151-230章 | 三段到四段武师境 | 真力首次离体，进入中三段 |
| 第231-310章 | 四段到五段罡身境 | 成为全国大学生武道顶点并毕业 |

### 晋级写作约束

- 每次晋级都必须有明确的身体积累、技术难题、失败节点、领悟过程和公开验证。
- 晋级不是一个单独章节事件，而是该阶段故事长期积累后的结果。
- 陆寻可以提前创造战技，但不能提前拥有下一段的公共能力。
- 高一段对低一段存在稳定压制，越级只能依赖特殊条件，不能成为常态。
- 每次晋级必须同步兑现一个核心爽点，并推动人物关系或人生选择发生变化。

### 毕业后的组织路线

- 陆寻不加入传统豪门职业队，也不完全单干。
- 陆寻、苏晚晴和大学舍友、同学共同建立一个独立武道联合体。
- 联合体同时具备职业竞技队和武道实验室功能，不是传统俱乐部。
- 陆寻以个人职业巡回赛和自身段位晋升为核心。
- 武道同学可以参加团体赛事，苏晚晴负责战术分析、科研和训练系统。
- 其他伙伴分别承担医疗康复、赛事运营、法律事务和商业合作等职责。
- 前期收入来自比赛奖金、科研项目和商业合作。
- 六段以后，联合体可以承接城市救援和国家任务，自然进入更大的故事舞台。
- 联合体长期开发普通人也能负担的科学武道训练体系，挑战顶级训练资源被少数豪门和机构垄断的现状。
- 该路线必须保留大学伙伴，同时保证陆寻仍然是个人武道升级主线的绝对核心。

## 三、主角天赋

### 已确认方向

- 陆寻不使用系统。
- 陆寻不捡宝物。
- 陆寻不接受复制型武道传承。
- 陆寻不是依赖特殊能力晋级，而是百年难遇的纯武道天才。

### 天才的具体表现

1. 理解原理极快：别人记招式，陆寻能迅速理解招式为何这样发力。
2. 身体修正极快：第一次可以做错，但很少重复犯同一个错误。
3. 实战进化极快：交手过程中持续理解对手，迫使对手不断改变打法。

### 天赋限制

- 天赋加快理解，不能凭空增加身体强度和真力。
- 每一段仍然需要完成身体积累和公开晋级标准。
- 没见过、没练过、没亲身经历过的内容不能凭空掌握。
- 身体跟不上理解时，强行施展会受伤。
- 六段以后必须创造自己的武道道路，不能一直学习别人。

### 入学测试

- 陆寻的武道天赋在大学入学测试中立即轰动全校，不走长期隐藏路线。
- 他的基础身体数据普通，但陌生动作学习与原理理解成绩打破校史纪录。
- 教练演示一次基础动作后，他能完整复现，并在第二次主动修正发力顺序。
- 他还能发现演示者因旧伤形成的动作偏差。
- 苏晚晴负责记录测试数据，是第一个确认设备没有出错、陆寻确实是怪物天才的人。

## 四、全民武道世界

### 基本原则

- 武道不需要觉醒。
- 人人都可以学习武道，但只有少数人能够进入正式段位。
- 武道类似现实中的体育与职业竞技：全民可学，专业分流。
- 人体通过训练产生真力；一至三段真力在体内运行，四段开始离体，六段由武道意志统合真力形成武道真身。

### 教育与组织

1. 中学武道课：体育必修内容，教授站姿、呼吸、发力和自我保护。
2. 大学武道学院：正式专业，培养职业武者、教练、军警和赛事选手。
3. 公共武道中心：面向全校学生，任何专业都能选修课程并参加段位考核。
4. 校武道队：代表学校参加大学联赛，不限制学生专业，只看实力。
5. 武道社：学生兴趣组织，负责日常交流和新人活动，不等于正式校队。
6. 智能武道实验室：通过动作捕捉、战术建模和训练数据分析辅助校队与科研。

### 武道的历史发展

1. 战争武术阶段：武道主要服务战争与治安，流派封闭，由少数家族掌握。
2. 段位标准化阶段：现代国家建立九段认证制度，将武道纳入学校、军队、警务和职业体系。
3. 科学武道阶段：大学结合医学、计算机、历史和训练数据研究武道，使经验传授逐步变成可验证的现代学科。

### 无怪物世界中的武道作用

- 一至五段主要参与校园与职业赛事、警务、安保、救援、教学和科研。
- 五段是普通职业武者能够达到的顶点，也是全国大学生武道的最高层级。
- 六至七段承担城市级战略职责，包括制止高段武者犯罪、大型灾害救援、保护关键设施和处理城市级武道冲突。
- 普通武器难以稳定限制宗师，因此高段武者主要由同层级武者制衡。
- 八段武王属于国家战略威慑力量。武王跨境会被视为重大军事行动。
- 当前时代可以没有九段武圣，各国依靠少数武王维持国际平衡。
- 陆寻冲击九段时，核心问题不只是个人晋级，还包括国家级力量究竟属于个人、国家还是全人类。

## 五、陆寻与历史系

### 为什么报考历史系

- 陆寻高中时身体数据普通，县城中学缺少专业武道设备，因此从未发现他的悟性天赋。
- 他真正擅长并喜欢历史，尤其关注战争胜负、武术流派兴衰，以及普通人在时代变化中的选择。
- 他原本计划读历史、考研，进入博物馆、档案馆或研究机构。
- 北渊大学的新生测试第一次加入陌生动作学习能力评估，因此发现了他的武道天赋。

### 为什么保留历史系学籍

- 历史不是陆寻的退路，而是早于武道存在的真实兴趣。
- 他不愿因为突然出现的天赋，就否定过去十八年的自己。
- 他不希望自己只成为一件用于比赛或战争的武器。
- 他将研究历代武道流派为何兴起、为何有效、又为何被淘汰，并把历史视角转化为建立个人武道体系的优势。
- 陆寻拒绝完全转入武道学院，以历史系学生、跨院特殊培养生和校队成员三重身份成长。

## 六、女主苏晚晴

### 人物路线

- 苏晚晴以战术分析和科研辅助为主，不走高段战斗武者路线。
- 她是计算机系学生，也是智能武道实验室的战术分析员。
- 她擅长动作捕捉、对手建模、训练数据分析和风险预测。
- 她的长期目标是建立普通家庭也能使用的低成本武道训练系统。
- 她的成长线是从学生分析员成为国家级武道科研专家。
- 她不是主角的外挂：她能提供数据和方案，但理解、训练和突破仍由陆寻完成。

### 与陆寻的关系

- 两人在新生武道测试中正式建立联系。
- 苏晚晴最初因异常测试数据跟踪陆寻训练，两人成为固定合作搭档。
- 陆寻帮助苏晚晴突破现有武道研究理论，苏晚晴帮助陆寻把直觉转化为可验证的训练方案。
- 第一卷前半成为固定搭档，中后期产生明确好感，卷末正式确定恋爱关系。
- 后续重点写恋爱后的磨合、职业选择和相互尊重，不用数百章反复暧昧。

## 七、故事线设计顺序

### 全书核心冲突

- “武道知识普及”与“高段力量管控”只作为后期世界背景和事件来源，不作为全书反复辩论的思想主题。
- 苏晚晴希望建立普通家庭也能负担的科学武道训练体系，陆寻用自身成长验证并完善这套体系。
- 训练体系越成功，普通人的晋级机会越多，传统豪门和国家机构对高端培养资源的垄断越弱。
- 与此同时，高段武者数量增加也会带来犯罪、城市安全和国家军备竞赛风险。
- 这些问题通过比赛资格、训练资源、赛事规则、城市任务和国家队选拔等具体事件表现，不安排大段理念争论。
- 全书核心阅读体验始终是热血战斗、段位突破、天才争锋、爽点兑现和穿插其中的恋爱日常。
- 后期格局扩大也必须服务战斗和升级，不把故事写成政治或社会议题讨论。

### 同代天才群像

- 全书安排多位与陆寻长期竞争的同代天才，而不是只有一个对手。
- 每位天才必须拥有不同的武道风格、核心欲望、成长上限和对武道未来的立场。
- 他们会在大学、职业、宗师和国家阶段反复出现，并随时间独立成长。
- 旧对手不能在战败后自动退场，也不能全部沦为主角追随者。
- 有人会成为伙伴，有人会坚持竞争，有人会改变道路，也有人会停在自己的天赋上限。
- 陆寻是唯一最终冲击九段的人，但其他天才必须在各自阶段拥有真正高光。
- 长期天才角色控制在少而精的范围，其他对手按分卷需要出现，不强行让所有对手承载宏大立场。

已确认四名长期天才的角色功能，姓名与具体战技等分卷时再确定：

1. 学院派枪道天才：国家重点培养，基础近乎完美，是陆寻最稳定的长期竞争对手。两人主要争夺同代最强，不承担宏大理念辩论。
2. 体魄型拳道天才：陆寻的大学舍友，身体天赋惊人、性格豪爽。毕业后参与建立武道联合体，既是兄弟也是会认真争胜的对手。
3. 剑道天才：其他顶级大学的女性选手，剑法精准、性格干脆。与陆寻长期交手且没有感情暧昧，并与苏晚晴建立独立友谊。
4. 身法型天才：职业赛阶段登场，速度极快、打法华丽、性格张扬，后期可以成为国家队队友。

国际篇和高段阶段根据每卷需要增加阶段性强敌，不要求所有阶段对手长期留场。
